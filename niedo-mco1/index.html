+

Separating concerns with internal <style> tags and an external stylesheet — using IDs, classes, and element selectors to apply organized, reusable styling across the page.

activity3/index.html
↗ Open
📄 activity3 / index.html
<!DOCTYPE html>
<html>
<head>
  <title>About Myself</title>

  <!-- External CSS -->
  <link rel="stylesheet" href="style.css">

  <!-- Internal CSS -->
  <style>
    h1 {
      color: purple;
      text-align: center;
    }
    #intro {
      font-style: italic;
      color: pink;
    }
    .hobby {
      color: lightseagreen;
      font-size: 18px;
    }
  </style>
</head>
<body>
  <h1>Hi! I'm Shailyn B. Niedo</h1>
  <h2 id="intro">Aspiring 2D Animator</h2>
  <h3 class="section">About Myself</h3>
  <p class="text">
    Hi my name is Shailyn, I'm 22 years old from Sta.Margarita Samar.
    I live in my parents House. I have 3 siblings and a loving family.
    I want to graduate so that I can pursue my dreams and help my family.
    Most of the time I love my dog and playing with her if I have free time.
  </p>
  <h3 class="section">My Hobbies</h3>
  <ul>
    <li class="hobby">Editing</li>
    <li class="hobby">Dancing</li>

  </ul>
</body>
